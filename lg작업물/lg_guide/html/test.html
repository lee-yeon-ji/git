냉장고 유형

냉동 설정온도 표시창에 나타난 해당 오류 코드를 아래에서 선택하세요.

냉동 설정온도 표시창에 나타난 해당 오류 코드를 아래에서 선택하세요.

이 콘텐츠는 공용으로 제작되어 구입한 제품과 다른 이미지나 내용이 포함되어 있을 수 있습니다.

IF Error Code

LED 램프 표시창에 [E IF] 표시되는 경우, 제빙실 팬모터 이상 에러코드로 제품 점검이 필요합니다.

※ 고객센터(1544-7777)로 문의해 주세요.

  출장 서비스 예약 | LG전자 고객지원 (lge.co.kr)

FF Error Code

LED 램프 표시창에 [ER(E) FF] 표시되는 경우, 냉동실 팬모터 이상 에러코드로 제품 점검이 필요합니다.

※ 제품 이상으로 발생하는 증상은 간단한 조치로 해결이 어렵습니다. 고객센터(1544-7777)로 문의해 주세요.

이렇게 해보세요.

냉동실 팬 모터이상 또는 팬 모터 주변에 성에나 얼음으로 인해 팬 모터가 구속된 경우에도 발생할 수 있습니다.

➔ 이런 경우에는 제품의 전원코드를 분리 후 냉동/냉장 도어를 완전히 열어 둔 상태에서 최소 8시간 이상을 기다리면 얼음(성에)이 녹게 됩니다.

만약 내부 식품 때문에 걱정이라면, 서비스 접수를 통해 점검을 받아보시는 것을 권유드립니다.

1. 식품을 보관할 여유분의 냉장고가 있다면 식품을 이동시켜 보관해주세요.

2. 제품의 전원 코드를 분리해 주세요.

3. 냉동/냉장 도어를 모두 활짝 열어 둔 상태에서 얼음(성에)이 녹는데 최소 8시간 이상을 기다려주세요.

4. 성에가 녹은 후 여분의 물을 흡수할 수 있도록 제품 안팎에 마른 수건을 이용해 내부에 남아있는 물기를 제거해주세요.

5. 전원을 다시 연결하고 냉장고의 냉기가 정상적으로 돌아올 때까지 24~48시간 정도 소요됩니다.

※ 만약, 동일 동일 점검코드가 지속되는 제품 이상으로 인한 문제일 수 있으므로 적절한 진단을 위해 서비스접수가 필요합니다.

CF Error Code

LED 램프 표시창에 [ER(E) CF] 표시되는 경우, 기계실 팬모터 이상 에러코드로 제품 점검이 필요합니다.

ER(E) CF 에러

➔ 냉장고 뒷면 기계실 냉각 모터[컴프레서] 열을 방출시켜주는 팬 모터 이상으로 동작하지 않을 때 표시창에 CF에러가 나타납니다.

보통은 팬 모터 고장 시에 발생하지만 간혹 배선 단선이나 메인보드 문제로 발생하기도 합니다.
전원을 껐다가 다시 켜면 에러가 사라지고 3시간 후에 다시 나타납니다.

※ 만약, 동일 점검코드가 반복된다면 정확한 점검을 위해 전문 서비스매니저의 확인이 필요합니다.

E CO(통신) 에러

➔ 메인보드와 디스플레이(표시창) PCB 간에 통신이 안될 경우 표시창에 CO 표시가 나타납니다.

통신이c 안되는 이유는 메인보드나 디스플레이 PCB 고장도 있지만 연결되는 배선이 단선되어 나타나는 경우도 있습니다.
정확한 점검을 위해 서비스접수가 필요합니다.

CH Error Code

냉장고 신규 설치 또는 이전 설치 후, [ER(E) CH/CL] 에러코드가 떴나요?

[ER(E) CH/CL] 에러코드는 냉장고 온도불량 냉기약함 감지 시 나타나는 점검 코드로, 사이클(냉매누설, COMP불량, 밸브불량 등)에 문제가 있을 수 있으니 적절한 진단을 위해 서비스접수가 필요합니다.

제품 설치 시 전원 인가 24시간 이후 냉기약함 감지 시 표시창에 냉장고 온도불량 증상을 알려주는 점검 코드입니다.

※ 고객센터(1544-7777)로 문의해 주세요.

  출장 서비스 예약 | LG전자 고객지원 (lge.co.kr)

AS Error Code

E AS(자동문 리드 스위치불량) r AS 또는 L AS(자동서랍 우측/좌측 리드스위치 불량)에러

E AS : 자동 문 모터 작동 시 리드 스위치 열림 상태 일 경우 나타나는 증상입니다. (리드 스위치 결선 불량)

R AS/L AS : 자동 서랍 우측/좌측 작동 시 리드 스위치 열림 상태 일 경우 나타나는 증상입니다. (리드 스위치 결선 불량)

리드스위치 불량으로 자동 서랍 모터 작동 시 리드스위치 닫힘 상태일 경우 나타나는 증상입니다. (자동 서랍 모듈 결선 검사, 모터 불량, 홀센서 불량)

해당 증상은 전문 서비스매니저 방문 점검이 필요합니다.
서비스 접수를 신청해 주세요.

O FF Error Code

이런 원인/증상이 있나요?

OFF 표시는 제품 판매 전에 매장에서 진열하는 기능으로, 내부 부품은 동작하지 않고 전원과 표시창 등 일부 기능만 동작시키는 특수 기능이에요.
표시창에서 조작하는 온도조절 버튼이 도어를 열고 음식물을 빼내는 과정에서 길게 잘못 눌러졌거나, 물티슈/젖은 수건 등으로 표시창을 닦는 과정에서 일부 버튼들이 눌러져 명령키로 인식하면 OFF 표시가 나타날 수 있어요.

이렇게 해보세요.

전원코드를 뺐다 꽂거나 차단기를 내렸다가 10초 정도 후 올려주세요.
이렇게 관리해 주세요.
1. 전면 표시창에 물기나 이물이 있다면 마른 수건으로 깨끗이 닦아주세요.
2. 온도 조절 버튼을 자주 사용하지 않으신다면 잠금/풀림 버튼을 2초 이상(약 3초) 길게 눌러 잠금으로 설정해 주세요.
3. 표시창 반대 우측 문을 열구 음식물을 꺼내신다면 왼손이 표시창을 누르지 않도록 주의해 주세요.

CF Error Code

LED 램프 표시창에 [C(대문자) 또는 E CF] 표시되는 경우, 기계실 팬 모터이상 에러코드로 제품 점검이 필요합니다.

C(대문자) 또는 E CF (기계실 팬모터이상) 에러

➔ 냉장고 뒷면 기계실 냉각 모터[컴프레서] 열을 방출시켜주는 팬 모터 이상으로 동작하지 않을 때 표시창에 C(대문자) 또는 E CF (기계실 팬모터이상) 에러가 나타납니다.
보통은 팬 모터 고장 시에 발생하지만 간혹 배선 단선이나 메인보드 문제로 발생하기도 합니다.
전원을 껐다가 다시 켜면 에러가 사라지고 3시간 후에 다시 나타납니다.

※ 만약, 동일 점검코드가 반복된다면 정확한 점검을 위해 서비스접수를 신청해주세요.

FF Error Code

LED 램프 표시창에 [Er(E) FF/F] 표시되는 경우, 냉동/냉장실 팬 모터이상 에러코드로 제품 점검이 필요합니다.

Er(E) FF / F에러

➔ 냉동실 팬 모터 이상 또는 정상 동작하지 않을 때 발생하는 에러입니다.
또는 팬 모터 주변 얼음(성에)으로 인해 팬 모터가 구속된 경우에도 발생할 수 있어요.

이런 경우에는 제품의 전원코드를 분리 후 냉동/냉장 도어를 완전히 열어 둔 상태에서 최소 8시간 이상을 기다리면 얼음(성에)이 녹게 됩니다.
만약 내부 식품 때문에 걱정이라면, 서비스 접수를 통해 점검을 받아보시는 것을 권유드립니다.

1. 식품을 보관할 여유분의 냉장고가 있다면 식품을 이동시켜 보관해주세요.

2. 제품의 전원 코드를 분리해 주세요.

3. 냉동/냉장 도어를 모두 활짝 열어 둔 상태에서 얼음(성에)이 녹는데 최소 8시간 이상을 기다려주세요.

4. 성에가 녹은 후 여분의 물을 흡수할 수 있도록 제품 안팎에 마른 수건을 이용해 내부에 남아있는 물기를 제거해주세요.

5. 전원을 다시 연결하고 냉장고의 냉기가 정상적으로 돌아올 때까지 24~48시간 정도 소요됩니다.

※만약, 동일 점검코드가 반복된다면 정확한 점검을 위해 서비스접수를 신청해주세요.

CO Error Code

LED 램프가 깜박거리거나, 표시창에 [c(소문자) 또는 E CO] 표시되는 경우, 통신이상 에러코드로 제품 점검이 필요합니다.

[c(소문자) 또는 E CO] (통신이상) 에러

➔ 메인보드와 디스플레이(표시창) PCB 간에 통신이 안될 경우 표시창에 점검코드가 나타납니다.

통신이 안되는 이유는 메인보드나 디스플레이 PCB 고장도 있지만 연결되는 배선이 단선되어 나타나는 경우도 있습니다.

※ 만약, 동일 점검코드가 반복된다면 정확한 점검을 위해 서비스접수를 신청해주세요.
